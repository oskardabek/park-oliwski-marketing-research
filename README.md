# Park Oliwski Coffee Shop — Badania Marketingowe
Kompletny projekt badawczy: koncepcja kawiarni dla lokalizacji w parku.
## Najważniejsze
- Badanie CAWI preferencji klientów (N = 54): wystrój, menu, ceny
- Analiza bazy kaw specialty (pochodzenie, profile sensoryczne)
- Drzewa klasyfikacyjne do identyfikacji kryteriów wyboru kawy premium
- Pełna koncepcja biznesowa: „Kawa dnia", analiza cenowa, koncept wizualny (fasada + wnętrze), menu
## Pliki
- `raport.pdf` — pełny raport z badań
- `ankieta.docx` — kwestionariusz CAWI
## Info
Projekt akademicki (Badania Marketingowe, Uniwersytet Gdański).
